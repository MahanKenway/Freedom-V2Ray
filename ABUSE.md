# Abuse Policy

Freedom-V2Ray is intended for research, testing, automation, and lawful network configuration management.

Users must not use this project to:

- gain unauthorized access to systems;
- scan or attack third-party networks;
- evade access controls unlawfully;
- distribute stolen credentials;
- perform denial-of-service attacks;
- violate applicable laws or service terms.

The maintainers do not operate or control third-party endpoints listed by external sources.

If you are the owner of a server or source listed in this project and want it removed, open an issue or contact the maintainer with:

- the affected URL or identifier;
- proof of ownership if necessary;
- the requested action.

Reported data may be removed without prior notice.
